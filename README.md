# TimerPlugin
プラグイン制作の勉強で作ったシンプルなタイマー
## これはなに？  
これは私がプラグイン制作の勉強で作成したシンプルなタイマープラグインです。  
やる気の関係上、停止コマンドは付けておりません。  
## 使い方
`/timer set <seconds>` です。
